Firebase Setup Required
The app needs a Firebase project to store data and handle login. This is a one-time setup — employees never need to touch it after deployment.
1
Go to console.firebase.google.com → create a new project
2
Enable Authentication → Google Sign-In and add kubixmedia.co.uk as an authorised domain
3
Enable Firestore Database and paste the security rules from the plan
4
Copy your Firebase config → find FIREBASE_CONFIG at the top of the <script> in this file → replace the placeholder values
5
Deploy to Netlify — done. Employees sign in with Google from that point forward.
